Arduino is an easy-to-use software and hardware. It consists of a circuit board, which can be controlled with Arduino IDE. Arduino is a beginer friendly microcontrollers, perfectly good for small projects due to its performance and price.

Raspberry Pi is more like a small size computer than a microcontroller. It supports a variety range of programming language like Python, C++,.. With its performance, it's capable of doing many complex tasks like browsing Internet, playing games
